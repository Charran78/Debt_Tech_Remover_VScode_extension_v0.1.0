## ✨ Características Clave

**Debt Tech Remover** es la implementación oficial en VS Code del framework **Sustainable Development Driven (SDD)**. Esta extensión transforma tu flujo de trabajo con IA en un proceso gobernado por una "Constitución" (`contract.md`).

- 🔍 **Agnostic Deep Grounding**: Escanea tu workspace local (Python, React, Go, Rust, etc.) para detectar el DNA tecnológico real y generar contratos ultra-específicos.
- 🛡️ **Gobernanza Proactiva**: Define límites de complejidad, patrones permitidos y reglas de seguridad que la IA debe respetar.
- 🤝 **Senior Partnership**: La IA no solo autocompleta; valida tus pasos contra el contrato y tiene derecho a veto ante propuestas que generen deuda técnica.
- 🔄 **Persistencia Total**: Gracias a la integración nativa con VS Code, tu sesión de chat y estado de generación se mantienen intactos aunque cambies de pestaña.

## 🚀 Instalación

### Desde el repositorio

1. Descarga el archivo `.vsix` desde la carpeta raíz de este repositorio (o desde los [Releases](https://github.com/Charran78/Debt_Tech_Remover_VScode_extension_v0.1.0/releases)).
2. En VS Code, abre la pestaña de **Extensions** (`Ctrl+Shift+X`).
3. Haz clic en el menú de tres puntos (`...`) en la esquina superior derecha.
4. Selecciona **Install from VSIX...** y elige el archivo descargado.

## ⚙️ Configuración

### 1. Gemini API Key

Para que el "Senior Partner" cobre vida, necesitas una clave de Gemini:

1. Obtén tu clave en [Google AI Studio](https://aistudio.google.com/app/apikey).
2. En VS Code, abre la paleta de comandos (`Ctrl+Shift+P`).
3. Busca: `Debt Tech Remover: Configurar Gemini API Key`.
4. Pega tu clave. Se guardará de forma segura en el almacenamiento cifrado de VS Code.

### 2. Licencia Pro (Opcional)

Si tienes una licencia de Gumroad:

1. Usa el comando `Debt Tech Remover: Activar Licencia Pro` desde la paleta de comandos.

## 🧠 Filosofía SDD (Sustainable Development Driven)

Este proyecto nace para resolver el problema de la "deuda técnica generada por IA".

- **Contract-First**: Primero se definen las reglas en el `contract.md`.
- **Auditoría en Tiempo Real**: Toda sugerencia de la IA es auditada contra la Constitución del proyecto.
- **Transparencia**: Sabes exactamente en qué se basa la IA para darte consejos.

## 🔒 Seguridad

- **Privacidad Local**: El escaneo de workspace se realiza localmente. Solo se envía a la API de Gemini el contexto necesario para el grounding.
- **Secret Storage**: Tus claves API nunca se guardan en texto plano; usamos la API nativa de VS Code para el manejo de secretos.
